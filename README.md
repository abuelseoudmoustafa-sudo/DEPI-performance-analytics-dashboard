# 🎓 DEPI Performance Analytics Dashboard

A professional data analytics dashboard developed for the **Digital Egypt Pioneers Initiative (DEPI)** to monitor and evaluate:

* Student attendance & engagement
* Training provider performance
* Group & track distribution
* Overall program insights

---

## 📊 Key Features

### 👨‍🎓 Students Performance

* Attendance tracking (Present, Absent, Late, Excused)
* At-Risk student identification
* Engagement segmentation (High / Medium / Low)
* Performance breakdown by track

### 🏢 Providers Performance

* Ranking of training providers
* Attendance rate comparison
* Identification of top and low performers

### 📈 Overview Dashboard

* Total sessions analysis
* Attendance trends over time
* Key performance indicators (KPIs)

---
